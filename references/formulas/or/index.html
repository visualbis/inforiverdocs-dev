OR

An OR statement returns TRUE if at least one of the arguments is TRUE and returns FALSE when all the arguments evaluate to FALSE.

Syntax#

OR ( logical_test1, [logical_test2], ... )

Arguments#

  • logical_test1 â€“ The first condition to be evaluated. This is a required argument.
  • logical_test2, â€¦ â€“ The additional conditions to be evaluated. These are optional arguments.

Return value#

Either True or False.

Remarks#

The OR function accepts multiple conditions.

Example#

Consider a scenario in which if the number of licenses is greater than 500 or the usage period is greater than 24 months, then the price should be $20 else $25.

To achieve this, the formula should be:

IF(OR(Num_of_Licenses>500,Duration<24),20,25)
