IF

An IF statement returns a value depending on whether the condition that is being evaluated is TRUE or FALSE.

Syntax#

IF( logical_test, value_if_true, value_if_false )

Arguments#

  • logical_test: The condition to be evaluated.
  • value_if_true: The value to be returned if the condition is met. The value may be numeric, string, column name or another function.
  • value_if_false: The value to be returned if the condition is not met. The value may be numeric, string, column name or another function.

Return Value#

Either value_if_true or value_if_false.

Remarks#

  • Returns Blank when the formula is misspelled.

  • The IF function returns an Error (ERR!) in any of the following cases:

    • Either value_if_true or value_if_false or both arguments are missing.
    • The argument value_if_true or value_if_false are not of numeric or string or column data type.
    • Use of incorrect operator in logical_test.

Example#

Consider the Revenue and Cost of Goods Sold to follow the trend as shown below:

ParameterJanFebMarAprMayJunJulAug
Revenue100124137023455236
Cost of Goods Sold2525252525252525

Now, Gross Profit Margin can be computed using the IF function as: (Revenue-Cost of Goods Sold)/Revenue.

In case, Revenue has period value of 0, Gross Profit Margin should return 0.

To achieve this, the formula should be:

IF(Revenue==0,0,(Revenue-Cost of Goods Sold)/Revenue)

The result of the above formula is shown below:

ParameterJanFebMarAprMayJunJulAug
Gross Profit Margin75%80%82%09%44%52%31%

Excel Equivalent#

IF

Tip
Use SWITCH function instead of nesting multiple IF functions.
