ISNAN

The ISNAN function checks for error and returns a default value if a value in the cell is an error.

Syntax#

ISNAN (value1,value2)

Arguments#

  • value1 â€“ The cell reference that is to be checked for error values.
  • value2 â€“ The value which is to replace the error values.

Return value#

value2 in the place of Error.

Example#
