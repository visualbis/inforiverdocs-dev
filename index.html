FAQ

  • What are templates? Templates are formats already existing in Inforiver which can be applied to a report in one click. Financial, Spreadsheet, IBCS are some of the templates available in Inforiver.
  • Can I format the numbers in the visual? Yes. You can apply various number formatting (Scaling, decimal points, number format) to either entire visual, column, row or cell.
  • Can I add rows and columns inside the visual? Yes. Customizable rows and columns can be added in the visual. Learn more about the different types of rows and columns here
  • Can I add annotations in the visual? Yes. Notes can be added at cell, row or column level and can be displayed as a footnote if required.
  • Can I add charts inside the visual? Yes. There are many types of charts that can be added in the visual(part of the table) â€“ Bar, Bullet, Lollipop, Sparkline, waterfall. This is available in Premium and Standard Editions. You can also add only charts using Infocharts.
  • Can I write back the data from Inforiver to my data source? Yes, with Inforiver writeback users would be able to writeback
  • Can I use my custom theme for the visual? Yes. The custom theme can be imported as a json file.
  • I would like to see my report by a Category in each page. Is this possible? Yes. Using Inforiver you can add page breaks based on the Category.
  • Which edition supports export to excel? The entire visual data can be exported to Excel using the Premium Edition. Premium Edition supports Export to PDF as well.
