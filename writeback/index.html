Writeback

The Writeback feature in Inforiver visuals provides you the ability to send data and formatting configuration from Inforiver to a Web URL. The data is exported in a .csv format.

Writeback Architecture

Configure Writeback in Inforiver#

  • In Power BI Desktop, select the Visualizations pane on the right-hand side.
  • From the Format section, Select Writeback.
  • Turn on the On toggle to enable writeback feature.

Enable writeback

  • In the Writeback URL box, enter URL of the backend data source to which Inforiver will post the data.

Performing Writeback#

  • Select the Export tab from the top ribbon bar.
  • Select Writeback.
  • Once the data is posted to the backend URL, a success message is displayed:
